## 🧱 Built With

- **HTML5**, **CSS3**, and **JavaScript**
- **Bootstrap 5** – for responsive design
- **AOS.js** – animation on scroll
- **PureCounter.js** – animated statistics
- **Custom assets** – icons, fonts, and imagery

## 🚀 How to Use

1. Clone or download the repository.
2. Open any `.html` file in a browser to preview the site.
3. Customize content, branding, and colors in the `assets/css/style.css` and HTML files.
4. Add your own business images in the `assets/img/` folder.
